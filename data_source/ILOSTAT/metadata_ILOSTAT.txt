To find the variables files, you can look at
download/www.ilo.org/ilostat-files/WEB_bulk_download/dic/indicator_*.csv
Prepend a frequency code to the variable name (A, Q, M).
Then the data is, for example for variable EAR_4MNP_NOC_NB annually, in 
indicator/EAR_4MNP_NOC_NB_A.csv.gz
The meaning of the column values with header some_header (classif1, 
note_indicator...) are in files
dic/some_header_en.csv

Replaces Laborsta which is in ../LABORSTA_ILO.
